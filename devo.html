Block Elements

This is a paragraph about the Block Elements

  • Always takes up the full width available
  • A block-level element always starts on a new line
  • The browsers automatically add some space (a margin) before and after the element.



Inline Elements

This is a paragraph about the Inline Elements

  1. An inline element only takes up as much width as necessary.
  2. An inline element does not start on a new line.
